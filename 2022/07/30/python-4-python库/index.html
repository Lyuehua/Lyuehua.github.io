python-4.python库

发表于 2022-07-30 分类于 language ， python

Python计算生态=标准库+第三方库

  • 标准库:随解释器直接安装到操作系统中的功能模块
  • 第三方库:需要经过安装才能使用的功能模块
  • 库Library、包Package、模块Module

以 turtle 库为例:

¶调用库的几种方法

¶form <库名> import <函数名>

¶import 调用库

<库名>.<函数名> 使用库函数

这种方法比较繁琐,每次调用都要带上<库名>.
但好处是特指此函数是库中的函数, 不会出现函数重名的问题

¶from <库名> import*

这种方式比较简洁,不需要每次都加上库名.来调用函数

虽然该方法调用函数比较简单,但是有发生函数重名的可能性

¶import <库名> as <库别名>

给调用的外部库关联一 个更短、更适合自己的名字

用这种方式,冗余的代码量最少,同时又防止了库重名的问题.
